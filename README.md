# Discord Rich Presence for Unity

Show your Unity project in Discord status while you work.

## Install

[![Add to VCC](https://img.shields.io/badge/Add%20to%20VCC-Install%20Now-blue)](vcc://vpm/addRepo?url=https://FR_KF_FR.github.io/Discord-Rich-Presence-for-unity-vrchatsdk/index.json)

Click the button above to add to VRChat Creator Companion.

## Manual Install

1. Open VRChat Creator Companion
2. Settings > VPM Settings
3. Add this URL: `https://FR_KF_FR.github.io/Discord-Rich-Presence-for-unity-vrchatsdk/index.json`
4. Install "Kodex Discord Rich Presence"

## Setup

1. Go to [Discord Developer Portal](https://discord.com/developers/applications) and create an application
2. Enable Rich Presence
3. Copy the Application ID
4. In Unity, go to **Kodex Tools > Discord Rich Presence**
5. Paste your Application ID
6. Add the DiscordRichPresence component to a GameObject in your scene

## Features

- Shows project name and current scene in Discord
- Shows Unity version
- Auto-updates when you change scenes
- Customizable text and images
- Works with VRChat projects
